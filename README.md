A beginner-friendly Python script that demonstrates fundamental programming concepts including variables, data types, arithmetic operations, and string manipulation.

## 📋 Overview

This repository contains a simple yet comprehensive Python script that covers the essential building blocks of Python programming. Perfect for beginners who are just starting their Python journey!

## 🎯 Topics Covered

### 1️⃣ Variables and Data Types
- String variables
- Integer variables  
- Float variables
- Boolean variables
- Type checking with `type()` function

### 2️⃣ Basic Calculator Operations
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Integer Division (`//`)
- Modulus (`%`)
- Exponentiation (`**`)

### 3️⃣ String Manipulation
- Converting case (`.upper()`, `.lower()`)
- String length (`len()`)
- String slicing
- String reversal
